# anglog — MVP 実行計画 / 進捗管理

**ゴール:** MVP機能全部（釣果CRUD ＋ 地図 ＋ 天気 ＋ 最低限UI）。公開デプロイは対象外（MVP後）。
**想定ペース:** 平日 1.5h ＋ 休日多め ＝ **週 ~11〜12h**。
**総見積り:** 残 ~32h（学習/デバッグ込み）→ **最短 3週間・完了目安 2026-07-27**。
**基準:** 開始 2026-07-07（火）。期日は無いが、この計画線を「巻き/遅延」の物差しにする。
（7/6は一覧ページ作成で消化済みのため、計画は翌日開始）

> 体制はレビュアー制（本人が全コード）。見積りは初学者ペース込み。詰まったら1タスク=1レビューで刻む。
